# JZZ1 主词 + 配件词长尾组合图谱 2026 Q2

这份文件专门回答一个问题：

`哪些主产品词，可以和哪些配件词 / 规格词 / 执行词组合，形成更有价值、更容易转化的长尾词？`

这里的“价值”不是拍脑袋，而是基于 4 个维度判断：

- 是否贴近真实 RFQ
- 是否能自然承接到现有页面
- 是否有更高询盘质量
- 是否不容易沦为低质量配件堆词

## 1. 组合原则

### 最高价值的组合，不是“主词 + 任意配件词”

而是：

- 主词 + 影响报价的规格词
- 主词 + 影响性能的部件词
- 主词 + 影响交付执行的 OEM / packing / label / tooling 词

例如：

- `aluminum windows with low-e glass`
- `thermal break aluminum windows with epdm gasket`
- `custom aluminum extrusion profiles with tooling support`
- `oem aluminum windows with project labels`

这些词更接近真实采购动作。

## 4. 暂不建议优先的组合方式

- 主词 + 过于泛化的配件词
- 主词 + 用户不会单独搜索的词
- 主词 + 无法自然承接到页面正文的词

例如：

- `aluminum windows gasket supplier`
- `aluminum doors accessories china`
- `profiles hardware factory`

这些词容易把页面拉向低质量泛流量，不利于网站整体权重积累。
